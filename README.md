## General info
This is my personal project created on **Rails 7.1**. The name of site is Chatofy. I created this as a part of my rails learning journey.

## Features
- It includes the [CRUD](https://en.wikipedia.org/wiki/Create,_read,_update_and_delete) actions for -***users***.
- It implements ***user authentication*** using [**Devise**](https://rubygems.org/gems/devise/versions/4.9.3?locale=en).
- Users can view other users on the site and can ***join and create*** **public and private rooms** and chat with others.
- Users can also create and view rooms as per category. 
- For database it uses -***SQLite3***.

## Technologies
- The techstack is:
